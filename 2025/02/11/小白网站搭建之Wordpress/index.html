准备步骤

下载工具

在开始使用WordPress之前，您需要准备一些必备的工具：

  • 小皮面板（推荐，但可以不用）：更简单的部署数据库和本地环境。
  • 主机与域名：用于安装和运行WordPress网站的空间。
  • 推荐使用最新版的Google Chrome、Firefox或Safari浏览器。

安装WordPress

  1. 访问WordPress官方网站（www.wordpress.org），下载最新版本的WordPress核心文件。
  2. 解压下载的文件，将解压后的所有文件上传到你的网络主机的公共html文件夹。
  3. 创建MySQL数据库，并记录下数据库名、用户名和密码。
  4. 在浏览器中打开你的网站地址，按照提示完成安装。

小皮面板

启动数据库和Apache
![[Pasted image 20250211130610.png|400]]

  • 点击“网站”，创建网站，里面内容随意写
  • 点击“数据库”，创建数据库，新建一个然后设置自己的密码
  • 打开刚创建网站的“根目录”，然后把wordpress的压缩包里面所有内容复制到这个根目录（原来的文件都不要）
    ![[Pasted image 20250211131756.png|300]]
  • 然后在浏览器输入网址，打开网站检查下是否跳转到wordpress
  • 跳转到wordpress就成功啦！

使用wordpress

安装并登录WordPress后，你会看到一个实用的仪表盘界面。左侧列表包含了所有WordPress网站的重要配置和管理选项，例如创建新文章、添加媒体、控制菜单等。

创建内容

  1. 添加文章：在仪表盘左侧导航栏点击“文章”→“新建”，进入新建文章页面。你可以在这里写文章标题和正文，进行排版，设置特色图像等。

  2. 添加页面：过程与添加文章相似，只需点击“页面”→“新建”即可。

我的使用感想

  • wordpress内存容量很大，网速也卡（可能要梯子），导致我每次用都很糟心
  • 虽然wordpress看上去一键操作和模板，但是灵活性还是很有限，代码也很难调整，过分依附平台
  • 文章我常用markdown写文章，所以hexo更合适我的使用习惯
  • hexo的确门槛高，需要一定编程思维，但是用wp做网站不代表是帮你解决所有，仍然要自己去思考哪些模块和内容，如何去呈现，这些其实hexo学会之后灵活性更高
